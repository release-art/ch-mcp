ch-mcp · UK Companies House MCP server

This is a Model Context Protocol server exposing the UK Companies House register as read-only tools. Connect an MCP-capable client (ChatGPT, Claude Desktop, Cursor…) to this host to use it.

The MCP streamable-HTTP endpoint accepts POST at / and /mcp; this page is what you get when you GET it from a browser.
